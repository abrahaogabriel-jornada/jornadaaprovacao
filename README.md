# Cronograma 2026 (Web)

Este é um site/app simples (Next.js) cuja tela inicial é um calendário mensal.

## Rodar no seu computador (opcional)
1) Instale o Node.js (LTS)
2) Abra a pasta do projeto e rode:

```bash
npm install
npm run dev
```

Depois abra: http://localhost:3000

## Colocar no ar (recomendado: Vercel)
1) Crie uma conta no GitHub
2) Crie um repositório novo e envie os arquivos deste projeto
3) Crie uma conta no Vercel e clique em **New Project**
4) Selecione o repositório e clique em **Deploy**

Pronto: você recebe um link do seu site.

## Backup
O app salva automaticamente no navegador (localStorage). Você também pode Exportar/Importar JSON.
